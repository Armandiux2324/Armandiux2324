# 🚀 Featured Projects

## **[Gym & Events Management Backend](https://github.com/Evodeps/gimnasio-backend.git)**
The backend of a comprehensive management system designed for a multifunctional fitness center and an event hall. The platform streamlines the administration of classes, events, users, reservations, and payments, improving the experience for clients, instructors, administrators, and super administrators.
The system includes role-based management, RESTful APIs, validations, media handling, scheduling functionalities, and scalable backend architecture focused on efficient business administration.

**Tech Stack:** Laravel, PHP, MySQL, REST API

## Zigzag Interactive Museum Access System

[Frontend](https://github.com/Armandiux2324/AccesosFrontend.git) •
[Backend](https://github.com/Armandiux2324/AccesosBackend.git)

A complete access and attendance management system developed for the Zigzag Interactive Museum. The platform allows visitor registration, access validation using QR codes, and administrative management through a responsive interface.
The frontend was developed with Angular, while the backend uses Express.js and JavaScript to provide API services and database communication.

**Tech Stack:** Angular, Express.js, JavaScript, MySQL
